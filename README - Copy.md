## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required!

## How to Use

- Browse products on the homepage
- Use the search bar to filter by brand, fabric, or style
- Click "order now" on any product to see details in a popup
- View product details for more information

## Uploading to GitHub

This project is ready to upload to GitHub. Just commit and push:

```bash
git add .
git commit -m "Initial commit"
git push
```

You can also host this directly on GitHub Pages by:
1. Going to your repository Settings
2. Navigate to Pages
3. Select the main branch as source
4. Your site will be live at `https://yourusername.github.io/repository-name/`
